## Carzilla Car Rental Website :-

### Used Technology :-

    - NodeJs
    - ExpressJs
    - MongoDB
    - Template Engine's ("ejs")

### How To Run :-

    - Open The Carzilla Folder In VS Code,
    - Then Open terminal using ctrl+~,
    - Then run this command
        - Step - 1
            - npm i
        - Step - 2
            - npm run dev
        - Step - 3
            - open any browser ("chrome,edge,firefox")

        - Step - 4
            - http://localhost:3000

    - For admin penal,
        - http://localhost:3000/admin
